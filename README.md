# 🌍 Global Events, Commodities and Currency Market Analysis

## Project Overview

This project analyzes the relationship between global events, commodity prices, and international currency markets using historical datasets for:

- Brent Oil
- Gold
- Silver
- SWIFT Global Currency Tracker

The objective is to understand how geopolitical events influence commodity prices and global payment currencies through exploratory data analysis and market research.

---

## Dataset

- Brent Oil Price Dataset
- Gold 100 Years Dataset
- Silver 100 Years Dataset
- SWIFT Global Currency Tracker

## Results

The analysis highlights:

- USD remains the dominant payment currency.
- Gold acts as a safe-haven asset during geopolitical uncertainty.
- Brent Oil prices exhibit high volatility during global crises.
- Commodity markets react strongly to geopolitical events.
